# 분야별 공식 출처 확인 방법

## 신뢰할 수 있는 출처 선택 기준

AI의 환각을 검증하려면 각 분야의 **공식 채널**에서 정보를 재확인해야 한다.

### 📋 법률·제도
- 국가법령정보센터(law.go.kr) — 법률·시행령·고시 원문
- 정부24(gov.kr) — 공식 정부 서비스 정보
- 각 부처 공식 홈페이지

### 🏥 의료·건강
- 국민건강보험공단(nhis.or.kr)
- 질병관리청(kdca.go.kr) — 감염병, 질병 정보
- 대한의사협회 공식 자료

### 📰 뉴스·사실
- 팩트체크넷(factcheck.snu.ac.kr) — 팩트체크 전문
- 연합뉴스 — 공식 통신사
- KBS 팩트체크

### 🎓 교육·학술
- 교육부(moe.go.kr)
- 한국교육과정평가원(kice.re.kr)
- RISS(학술논문) — 국내 학술지·학위논문 검색

### 💻 AI·기술
- 한국인터넷진흥원(kisa.or.kr)
- 과학기술정보통신부 공식 문서
- OpenAI 공식 문서 — 모델 사양, 제약사항

## 출처 신뢰도 판단 3가지 기준

✅ **신뢰할 수 있음**
- 기관 명칭 명확 + 날짜 표시 + 전문가 서명
- 공식 채널(gov.kr, 학술DB 등)에서 직접 확인 가능

❌ **의심 필요**
- 작성자 불명 + 날짜 없음 + 공유 링크만 있음
- 출처 기재 안 됨

⚠️ **AI 주의: "출처 제시 환각"**
- AI가 "출처: 00"이라고 말해도 **반드시 직접 확인 필요**
- AI는 출처 자체를 지어낼 수 있음
- 언급한 논문, 기관, 수치를 직접 검색해서 존재 확인
