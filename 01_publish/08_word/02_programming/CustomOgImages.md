# CustomOgImages

* `CustomOgImages`가 하는 일은 **링크 미리보기(Open Graph 이미지)를 생성하는 GitHub Action** 
* 생성 형식은 설정에 따라 다르며, 반드시 WebP만 사용하는 것은 아님
* Open Graph(OG) 미리보기 이미지를 자동 생성하는 GitHub Action
* **HTML 페이지를 SNS나 메신저에 공유할 때 표시되는 Open Graph 이미지를 생성하는 GitHub Action**
- **웹페이지 링크 공유 시 표시되는 미리보기(OG) 이미지를 자동 생성하는 GitHub Action**
