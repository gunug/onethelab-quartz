**CORS (Cross-Origin Resource Sharing)** is a browser security mechanism that allows a web server to explicitly specify which other websites (origins) are allowed to access its resources. It extends the browser's **same-origin policy** in a controlled way, enabling modern web applications to communicate securely with APIs hosted on different domains.

**CORS(Cross-Origin Resource Sharing)**는 웹 서버가 자신의 리소스에 접근할 수 있는 다른 웹사이트(출처/origin)를 명시적으로 지정할 수 있게 해주는 브라우저 보안 메커니즘입니다. 이는 브라우저의 **동일 출처 정책(same-origin policy)**을 제어된 방식으로 확장하여, 최신 웹 애플리케이션이 서로 다른 도메인에 호스팅된 API와 안전하게 통신할 수 있도록 합니다.
