
# 개요
* 가상계정에 play store 배포권한등을 설정하고 AI가 cli를 통해 앱의 출시등을 진행할 수 있도록 하는 절차
* Google Play Developer API v3(`androidpublisher`)가 전체 출시 워크플로를 커버
* 서비스 계정에 권한만 부여되어 있으면 빌드부터 출시까지 전 과정이 완전 자동화 가능

# 배포 계정 json
* play console 에서 다운로드 가능
* 새로운 앱을 추가하면 앱 권한 등록해야함
