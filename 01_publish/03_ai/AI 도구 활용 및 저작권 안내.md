## 🤖 AI 도구 활용에 대한 이해

이 수업에서는 **Google Gemini 같은 AI 도구를 적극적으로 활용**합니다. AI는:
- **학습 보조 도구**로서 아이디어 발상, 이미지 생성, 피드백 제공
- **창의적 사고의 확장**을 돕는 파트너
- **전문적 워크플로우의 일부**

AI를 활용하는 것은 **현대 디지털 미디어 제작의 표준 방식**이며, 이를 효과적으로 사용하는 능력이 중요합니다.

### ✅ AI 활용 시 주의사항

1. **AI 생성 콘텐츠는 학습 자료로만 사용**
   - 생성된 이미지/텍스트는 과제 제출용 임시 자료
   - 최종 결과물은 **자신의 판단과 수정**을 거쳐야 함

2. **AI의 한계 인식**
   - 생성된 내용이 항상 100% 정확하거나 창의적이지 않을 수 있음
   - 여러 번 수정 요청하면서 자신의 의도를 반영하는 과정이 중요

3. **윤리적 활용**
   - 타인의 저작물을 AI로 무단 복제하면 안 됨
   - 생성된 이미지/콘텐츠도 출처 표기 필요한 경우 명시

## 📋 저작권 및 위탁 관련 안내

### 🎨 과제 제출 시 명시해야 할 사항

각 과제 제출 때 **다음 정보를 포함**하면 좋습니다:

```
학번:
이름:
과제명:
AI 도구 활용 여부: ○ 예 / ○ 아니오
사용한 AI 도구: (예: Google Gemini, 기타)
AI로 생성된 콘텐츠: (예: 배경 이미지, 아이디어 기획 등)
```

### 📌 저작권 유의사항

1. **다운로드 이미지 저작권**
   - Google 검색에서 다운로드한 이미지는 개인 학습용으로만 사용
   - 상업적 이용 금지, 필요시 저작권자 허락 필요

2. **AI 생성 이미지 저작권**
   - Google Gemini로 생성한 이미지는 학습/교과 목적으로 사용 가능
   - 상업 용도 전 이용약관 확인 필수

3. **최종 포트폴리오 제출 시**
   - 다른 학생의 작품 무단 사용 금지
   - 참고한 자료가 있으면 반드시 출처 명시

## ⚠️ 모든 결정을 AI에게 위임할 때의 문제점

AI는 강력한 도구이지만, **모든 창의적 판단을 AI에게 맡기면 안 됩니다.** 다음은 AI에 완전히 의존했을 때의 문제점입니다:

### 🔴 **학습 능력 저하**

- **문제**: AI의 결과를 그대로 받아들이면, 자신이 직접 생각하고 판단하는 능력이 떨어짐
- **결과**: 나중에 AI 없이는 아무것도 할 수 없는 상황 발생
- **예시**: "AI가 색감을 정했으니 이게 맞다" → 색감 이론을 배우지 못함

### 🔴 **창의성 상실**

- **문제**: AI는 기존 데이터의 조합이므로 진정한 혁신적 아이디어 생성 불가
- **결과**: 모든 작품이 평범하고 획일적이 됨
- **예시**: 포켓몬 브로슈어를 만드는 5명이 AI에게 맡기면 5개 모두 비슷한 결과

### 🔴 **책임감 부재**

- **문제**: "AI가 이렇게 했어요"라고 책임을 미룸
- **결과**: 자신의 작품에 대한 애정과 소유권 부족
- **예시**: 평가에서 지적받으면 "AI가 잘못 만들었어요"라고 변명

### 🔴 **비판적 사고 능력 약화**

- **문제**: AI 결과물을 검토하고 개선하는 능력 부족
- **결과**: 틀린 정보나 어색한 부분을 못 알아챔
- **예시**: AI가 생성한 이미지에 논리적 오류가 있어도 받아들임

### 🔴 **기술 편차 증가**

- **문제**: 같은 도구를 사용해도 **효과적으로 사용하는 능력의 차이**가 큼
- **결과**: AI의 질문을 잘못 주면 결과도 나쁨
- **예시**: 상세한 프롬프트를 작성하는 학생 vs 한두 단어로 요청하는 학생의 결과 차이

## ✅ 올바른 AI 활용 방법

### 1️⃣ **AI는 파트너, 주인은 당신**
```
❌ "AI가 정해줄 때까지 기다리기"
✅ "내 의도를 AI에게 명확하게 전달하기"
```

### 2️⃣ **AI 결과 → 자신의 판단으로 검토 → 수정/개선**
```
생성 → 검토 → 의견 반영 → 재생성 → 최종 선택
(반복 과정이 중요!)
```

### 3️⃣ **"왜?" 질문하기**
```
❌ "AI가 파란색을 추천했으니 파란색으로"
✅ "왜 파란색이 좋을까? 다른 색도 한번 시도해볼까?"
```

### 4️⃣ **AI 없이 생각해보기**
```
AI 사용 전에 스스로 5분 생각하기
→ AI의 답과 자신의 생각 비교
→ 왜 다른지 분석해보기
```

## 🎯 이 과정에서 얻어야 할 것

| 항목 | 의미 |
|------|------|
| **AI 활용 능력** | 좋은 프롬프트 작성, 결과 검토 능력 |
| **판단력** | AI 결과를 평가하고 선택하는 능력 |
| **창의성** | AI 결과를 바탕으로 자신만의 개선 |
| **책임감** | 완성된 작품은 **내 작품**이라는 의식 |

**결론**: AI는 **시간을 절약해주는 도구**일 뿐, **당신의 사고와 판단을 대체할 수 없습니다.**

AI를 쓸수록 더 많이 생각하고, 더 자주 물어보고, 더 적극적으로 개선하는 학생이 되어야 합니다! 💪
