| 챗봇 | 회원가입 | 무료 | 특징 |
|------|--------|------|------|
| **Claude** | 불필수 | ✓ | 한국어 지원, 코드 설명 우수 |

| **🌟 Google Gemini** | 불필수 | ✓ | 멀티모달 지원, 이미지 분석, **Google 계정 연동** |

## 사용 챗봇
**Google Gemini** - Google 계정으로 바로 접속 가능, 추가 가입 불필요

---

## Google Gemini 사용법

### 접속 주소
🔗 **https://gemini.google.com**

### 간단한 사용 방법

1. **접속**
   - 브라우저에서 위 주소 입력
   - Google 계정으로 로그인 (또는 이미 로그인된 경우 바로 이용)

2. **질문하기**
   - 화면 하단 입력창에 질문 입력
   - "Enter" 키 또는 전송 버튼 클릭

3. **주요 기능**
   - 📝 텍스트 분석 및 설명
   - 🖼️ 이미지 업로드 및 분석
   - 💾 대화 저장 및 공유 가능
   - 🔄 이전 대화 이어서 계속 가능

4. **팁**
   - 이미지 파일 드래그앤드롭으로 업로드 가능
   - "새 대화" 버튼으로 새로운 주제 시작
   - 왼쪽 패널에서 이전 대화 내역 확인 가능

## 실습 예제

### 1️⃣ 📝 텍스트 분석 및 설명

**예제 질문:**
```
다음 코드를 설명해줘:

function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```

**학습 목표**: 복잡한 코드, 개념을 AI가 어떻게 쉽게 설명하는지 이해하기

### 4️⃣ 🔄 이전 대화 이어서 계속하기

**실습 방법:**
1. 왼쪽 패널에서 이전 대화 선택
2. 대화를 이어서 계속하거나 같은 주제로 깊이 있는 질문하기

**예제 상황:**
```

AI 답변 받음

그 다음 질문: "위 게임에 어떤 보스 캐릭터를 추가하면 좋을까?"
또는: "위 게임의 난이도 곡선을 설계해줄 수 있어?"
```

**학습 목표**: AI와의 연속적인 대화를 통해 아이디어 깊이 있게 발전시키기

## 추가 실습 내용

### 5️⃣ 🎨 이미지 생성 및 시각화

**실습 방법:**
1. Gemini에 이미지 생성을 요청하면 AI가 시각적 콘텐츠를 자동 생성
2. 생성된 이미지는 다운로드하여 게임 디자인이나 프로젝트에 활용 가능
