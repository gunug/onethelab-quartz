# vscode 설정
* vscode를 여러게 사용중 창을 겹치지 않도록 하는 vscode 설정 요청

제목 : 제목명 입력
색상 : 색상명 입력
1. vscode의 창을 다른 vscode의 창과 시각적으로 다르게 보이게 하기위한 조치
2. vscode의 설정파일 생성하고 다음의 내용을 기입
3. vscode의 제목을 설정
4. 프로젝트 제목만 있으면 됨 현재 열린창 이름을 추가로 표시하지 않음
5. 타이틀바 글씨색을 **색상**로 변경
6. 타이틀바 배경색을 **어두운 색상**로 변경
7. 파일 리스트(사이드바) 글씨색을 **색상**로 변경
8. 파일 리스트(사이드바) 배경색을 **아주 어두운 색상**로 변경
9. 파일 리스트에서 선택된 항목 글씨색은 가독성을 위해 **기본색상**으로 설정
