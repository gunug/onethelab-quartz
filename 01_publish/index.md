## One The Lab

One The Lab은 **인터랙티브 전시 솔루션** 회사입니다. 인쇄된 설명과 정적인 전시에서 벗어나, 이미지·사운드·영상과 Kinect·Leap Motion·Arduino 같은 인터랙션 장치로 관람객이 직접 체험하는 콘텐츠를 만듭니다.

- 🌐 회사 사이트: [onethelab.com](https://onethelab.com)
- ✉️ 문의: gh.cho@onethelab.com

## 무엇을 하나

| 분야 | 내용 |
| --- | --- |
| **Content Programming** | Unity3D 기반 VR·AR·2D·3D 콘텐츠 (Windows / Android) |
| **Physical Computing** | 3D 설계·프린팅, 레이저 커팅, PCB 설계, Arduino 제어 |
| **Device Remodeling** | 기계 장치 커스터마이징, 환경 맞춤 장비 제작 |
| **System Integration** | Serial·Socket·UDP·HID 연동, 하드웨어 호환성 해결 |
