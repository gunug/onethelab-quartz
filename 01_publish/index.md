# One The Lab

이 사이트는 Obsidian 노트를 [Quartz](https://quartz.jzhao.xyz)로 변환해 발행합니다.

- 발행 소스 폴더: `01_publish`
- 이 폴더 안의 `.md` 파일만 공개됩니다.
- 폴더 구조 = 사이트 구조.
- `[[위키링크]]`, 태그, 백링크 모두 지원.
- 비공개로 두려면 노트 맨 위 frontmatter에 `draft: true`.

## 시작하기

1. 발행할 노트를 `01_publish` 로 복사
2. 커밋 & 푸시 → GitHub Actions가 자동 빌드/배포
