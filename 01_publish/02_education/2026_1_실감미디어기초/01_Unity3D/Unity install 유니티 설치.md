# Unity 설치 가이드

Unity 게임 엔진을 설치하는 간단한 튜토리얼입니다. 대학생들도 쉽게 따라 할 수 있습니다.

---

## 📋 시스템 요구사항

### 최소 사양
- **OS**: Windows 10 이상, macOS 10.13 이상
- **RAM**: 4GB 이상 (8GB 권장)
- **저장공간**: 20GB 이상 (SSD 권장)
- **GPU**: 2GB 이상 VRAM (Nvidia 또는 AMD)
- **인터넷**: 설치 및 에셋 다운로드용

### 권장 사양
- **OS**: Windows 11, macOS 최신 버전
- **RAM**: 16GB 이상
- **GPU**: 4GB 이상 VRAM
- **SSD**: 설치 드라이브

---

## 1단계: Unity Hub 다운로드

### 1.1 공식 웹사이트 방문

**Unity 공식 다운로드 페이지:** https://unity.com/download

### 1.2 다운로드 시작

1. 웹사이트에서 자신의 OS에 맞는 버튼 클릭:
   - **"Download for Windows"** (Windows 사용자)
   - **"Download for Mac ARM64"** (Mac M1/M2 사용자)
   - **"Download for Mac Intel"** (Mac Intel 프로세서 사용자)

2. 설치 파일(`.exe` 또는 `.dmg`) 자동 다운로드 시작

---

## 2단계: Unity Hub 설치

### 2.1 설치 파일 실행

**Windows:**
1. 다운로드된 `UnityHubSetup.exe` 더블클릭
2. "설치" 버튼 클릭
3. 설치 완료 대기 (2-5분)

**Mac:**
1. 다운로드된 `UnityHub.dmg` 더블클릭
2. Unity Hub 아이콘을 Applications 폴더로 드래그
3. Applications에서 Unity Hub 실행

> ⚠️ **Mac M1/M2/M3/M4 사용자 주의**: 프로젝트 시작 시 진행률이 멈추거나 무한로딩이 발생하면 [[macOS Rosetta 설치 가이드]]를 참고하세요.

### 2.2 Unity Hub 실행

설치 완료 후 Unity Hub 시작

---

## 3단계: Unity Hub 로그인

### 3.1 계정 만들기 (처음 사용자)

1. Unity Hub 열기
2. **"Sign in"** 버튼 클릭
3. **"Create a new Unity ID"** 또는 **"Create account"** 클릭
4. 이메일, 비밀번호 입력
5. 인증 메일 확인 및 이메일 인증 완료

### 3.2 로그인

1. Unity ID 이메일 입력
2. 비밀번호 입력
3. **"Sign in"** 버튼 클릭

> **Tip**: 학생 이메일(`.ac.kr`, `.edu` 등) 사용 시 학생 라이선스 혜택 받을 수 있음

---

## 4단계: Unity 에디터 설치

### 4.1 설치 버전 선택

Unity Hub 왼쪽 메뉴에서 **"Installs"** 클릭

### 4.2 Unity 버전 추가

**"Install Editor"** 버튼 클릭

### 4.3 최신 LTS 버전 선택
![[Pasted image 20260306014442.png]]
* 6000번대 버전 설치하지 않도록 주의
```
권장: 2022 LTS 또는 2023 LTS
(최신 기능 + 장기 지원)
```

LTS(Long Term Support) 버전 선택:
- 더 안정적
- 장기 지원 제공
- 프로젝트에 권장

### 4.4 모듈 선택 (기본값 사용)

모듈 선택 페이지에서 **기본값으로 건너뛰기**

- 기본 선택되는 모듈들로 충분함
- Editor, Development Tools 등이 자동 선택됨
- 나중에 필요시 추가 모듈 설치 가능

### 4.5 설치 시작

**"Install"** 버튼 클릭

설치 시간: **15-45분** (인터넷 속도에 따라)
