Token 사용법

하드코딩 금지. 항상 CSS 변수로 참조한다.

/* ✗ 하지 말 것 */
.box{ background:#FFD400; }

/* ✓ 토큰 참조 */
.box{
  background: var(--hhg-yellow);
  color: var(--ink);
  padding: var(--s4);
  border-radius: var(--r-md);
  box-shadow: var(--sh-md);
}
