SVG 외부 파일 구조

파일 위치frame → svg/frame/ · insert → svg/insert/ · logo → svg/logo/
파일명 = idfrm_{quad}_bold_{name}.svg · full frm_full_bold_*.svg · ins_bold_{cat}_{name}.svg · logo_*.svg
모양 변경해당 .svg 1개만 단독 수정 → HTML 손 안 댐
새 파츠 추가① .svg 파일 생성 → ② HTML MANIFEST 배열에 id 한 줄 추가 (HTML 수정은 이 1곳뿐)
파츠 제거MANIFEST 에서 id 삭제 (파일은 둬도 무방)
svg 내부 규칙루트 <svg viewBox="-2 -2 28 28"> · 색 속성 금지 → 클래스만: frame .arc, center .ins/.ins-dot (색은 HTML CSS 변수가 입힘)
메타데이터 (파일 안)data-name = 버튼 표시명 (필수) · data-ic="x,y" = insert 중심 보정 (full frame 만, 선택)
